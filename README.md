## gulp-boiler-plate

### 개요

- javascript application을 더 편리하게 사용하기 위한 빌드시스템 gulp를 구성해놓은 boiler-plate 입니다.

### 사용법

- `git clone ${리포지토리주소}` 으로 boiler-plate를 복사합니다.
- `npm install` 명령어를 통해 의존 라이브러리를 설치합니다. (Node 20. 이상)
- `gulp` 명령어로 프로젝트를 시작합니다.
